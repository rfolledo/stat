Slide 2: What is the Binomial Test?

  • Definition: A statistical test used to determine if the proportion of successes in a sample differs from a hypothesized population proportion.
  • When to Use:
    • Binary outcomes (e.g., success/failure, yes/no).
    • Fixed number of trials (n).
    • Constant probability of success (p).
  • Example: Testing if a coin is fair (p = 0.5) based on flip results.

Slide 3: Key Concepts

  • Binomial Distribution:
    • Models the number of successes (X) in n independent trials.
    • Probability mass function:
      \[ P(X = k) = \binom{n}{k} p^k (1-p)^{n-k} \] where () is the binomial coefficient.
  • Hypotheses:
    • Null (H₀): The population proportion equals the hypothesized value (p = p₀).
    • Alternative (H₁): p ≠ p₀ (two-tailed), p > p₀, or p < p₀ (one-tailed).

Slide 4: Test Statistic

  • Formula:
    • For large samples, use the z-test statistic (normal approximation): \[ z = \frac{\hat{p} - p_0}{\sqrt{\frac{p_0 (1 - p_0)}{n}}} \] where:
      • () = sample proportion (x/n, x = number of successes).
      • (p_0) = hypothesized proportion.
      • (n) = number of trials.
  • Assumption: np₀ ≥ 5 and n(1-p₀) ≥ 5 for normal approximation.

Slide 5: Steps of the Binomial Test

  1. State Hypotheses:
    • Example: H₀: p = 0.5, H₁: p ≠ 0.5.
  2. Collect Data: Record number of successes (x) and trials (n).
  3. Calculate Test Statistic: Use the z-formula.
  4. Determine P-value: Compare z to the standard normal distribution.
  5. Make a Decision: Reject H₀ if p-value < significance level (α, e.g., 0.05).

Slide 6: Example Problem

  • Scenario: A coin is flipped 100 times, resulting in 60 heads. Is the coin fair (p = 0.5)?
  • Step 1: H₀: p = 0.5, H₁: p ≠ 0.5.
  • Step 2: x = 60, n = 100, ( = 60/100 = 0.6).
  • Step 3: Calculate z: \[ z = \frac{0.6 - 0.5}{\sqrt{\frac{0.5 \cdot 0.5}{100}}} = \frac{0.1}{\sqrt{0.0025}} = \frac{0.1}{0.05} = 2.0 \]
  • Step 4: P-value (two-tailed, z = 2.0) ≈ 0.0455.
  • Step 5: Since 0.0455 < 0.05, reject H₀. Evidence suggests the coin may not be fair.

Slide 7: Interpreting Results

  • P-value < α: Reject H₀, suggesting the sample proportion significantly differs from the hypothesized proportion.
  • P-value ≥ α: Fail to reject H₀, no significant evidence of a difference.
  • Caution: Normal approximation may not hold for small n or extreme p₀ values; use exact binomial test instead.

Slide 8: Applications

  • Quality Control: Test if defect rates match expected standards.
  • Medical Research: Assess if treatment success rates differ from a baseline.
  • Social Sciences: Evaluate survey response proportions (e.g., voter preferences).

Slide 9: Limitations

  • Requires independent trials and constant probability.
  • Normal approximation is less accurate for small samples (n < 30) or extreme proportions (p close to 0 or 1).
  • Alternative: Use exact binomial test for small samples (software like R or Python).

Slide 10: Conclusion

  • The binomial test statistic is a powerful tool for testing proportions.
  • Key steps: Define hypotheses, calculate z, compare p-value to α.
  • Use with caution for small samples or extreme proportions.
